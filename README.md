# Проект fasta_protein_sync
Для анализа большого количества белковых последовательностей из FASTA-файла. 

Программа:
- классифицирует аминокислоты по физико-химическим группам
- подсчитывает их количество в каждой последовательности и суммарно по всему файлу

Реализованы однопоточная, многопоточная и многопроцессная версии для сравнения производительности.

## Скопировать проект
```bash
git clone https://github.com/Uminelka/fasta_protein_sync.git
cd fasta_protein_sync
```

## Виртуальное окружение и зависимости
```bash
# Создание окружения
python -m venv venv
# Активация окружения
.\venv\Scripts\activate
# Установка зависимостей
pip install -r requirements.txt
```
